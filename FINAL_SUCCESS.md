All working and tested:

```bash
# Get all movies
GET /api/movies

# Get featured movies
GET /api/movies/featured

# Get trending movies
GET /api/movies/trending

# Get movie by ID
GET /api/movies/:id

# Search movies
GET /api/movies/search?q=term

# Register user
POST /api/auth/register

# Login user
POST /api/auth/login

# Get user's watchlist
GET /api/watchlist

# Add to watchlist
POST /api/watchlist/:movieId

# Remove from watchlist
DELETE /api/watchlist/:movieId

# Seed database (admin)
GET /api/seed-database
```
